# Contributing to GlowGuide

Thank you for helping improve GlowGuide. This project is an inclusive personal care knowledge base focused on skincare, bodycare, hygiene, and haircare education.

## Ways to Contribute

- Add beginner-friendly product explanations.
- Suggest routine improvements.
- Add ingredient notes or caution notes.
- Report unclear, unsafe, outdated, or duplicated content.
- Improve documentation, UI, accessibility, or code quality.

## Content Guidelines

- Use simple, non-judgmental, inclusive language.
- Keep content educational.
- Do not make medical claims or promise results.
- Avoid diagnosis or treatment instructions.
- Use safer wording such as "may help", "commonly used for", or "consider consulting a qualified professional".
- Include a source link where possible.

## Contribution Review

Public content suggestions should remain in a pending state until reviewed by a maintainer. Maintainers may approve, reject, or request changes.

## Local Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

## Pull Request Checklist

- [ ] The app runs locally.
- [ ] New content is beginner-friendly.
- [ ] Medical claims are avoided.
- [ ] Documentation is updated if behavior changed.
